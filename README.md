# STUDAID

STUDAID is a student web app that helps users discover useful places around campus, such as study spots, cafés, food places, gyms, and events.

## Target User
Students looking for useful and affordable places near their campus.

## Features

### Must Have
- Homepage with useful places
- Category buttons
- Place cards with name, category, and description

### Should Have
- Discover page
- Profile page
- Saved places section

### Could Have
- Search bar
- Filters by category

## Data Model

### Place
- id
- name
- category
- description

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

- Vercel
